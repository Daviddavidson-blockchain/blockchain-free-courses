## 📚 What is Blockchain?

Blockchain is a decentralized digital technology used to securely record transactions and data across multiple systems.

It is widely used in:
- Cryptocurrency
- Smart contracts
- Web3 applications
- Supply chain systems

---

## 🎯 Why Learn Blockchain?

Blockchain is one of the fastest-growing technologies with high demand across industries.

Learning blockchain can open career opportunities in:
- Web3 development
- Blockchain engineering
- Smart contract development
- Decentralized applications (dApps)
